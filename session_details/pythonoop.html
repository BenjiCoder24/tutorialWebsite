Introduction to Python OOP

This session covers the fundamentals of Object-Oriented Programming (OOP) in Python, focusing on classes, objects, and inheritance.

Objectives

  • Understand the basic concepts of OOP
  • Learn how to define and use classes and objects
  • Explore inheritance and its benefits

Materials Needed

  • Computer with Python installed or access to an online Python environment
  • Jupyter Notebook or Google Colab access
  • Access to the OOP tutorial repository

Expected Outcomes

Upon completion of this session, students will be able to:

  • Create simple classes and objects in Python
  • Utilize methods and attributes in classes
  • Implement inheritance to extend class functionality
